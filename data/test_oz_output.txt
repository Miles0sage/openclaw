The oz_executor.py file is a wrapper for the Oz CLI agent that executes AI tasks using Warp's multi-model orchestration (GPT-5.2, Claude 4.6, Gemini 3 Pro, etc.) with automatic model selection based on task complexity and priority. It provides a fallback execution chain from Oz to OpenCode to SDK, handles cost tracking and logging, and parses streaming JSON output from the Oz CLI to return structured results including text responses, tool calls, and estimated costs.
