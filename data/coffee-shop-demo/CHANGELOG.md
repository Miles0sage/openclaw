# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-05

### Added

- Initial release of Ember & Brew coffee shop template
- Full-viewport hero section with parallax background and animated text
- About section with two-column layout, story text, and key statistics
- Interactive menu with category filtering (Espresso, Drip & Pour Over, Cold Brew, Pastries)
- Photo gallery with hover overlays and zoom transitions
- Contact section with hours, location info, map placeholder, and contact form
- Sticky navigation bar with transparent-to-solid transition on scroll
- Mobile-responsive design with hamburger menu
- Intersection Observer scroll animations for menu items
- Dark warm color theme using CSS custom properties
- Google Fonts integration (Playfair Display + Inter)
- GitHub Actions workflow for GitHub Pages deployment
- Issue templates for bug reports and feature requests
- MIT License
- Contributing guidelines
