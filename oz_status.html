🔧 What We Fixed

Oz Ephemeral Container Bug FIXED
Oz containers are ephemeral — file changes vanished on exit. Execute phases now skip Oz entirely, go straight to OpenCode.
Code Review Severity Gating FIXED
Only blocked on "critical" severity, ignored "major" (7 issues passed through). Now blocks on both.
Oz Timeout 300s → 120s FIXED
61% of Oz execute phases timed out at 300s. Reduced to 120s — captures 95% of successes, saves ~3 min per timeout.
Circuit Breaker FIXED
Max 2 Oz failures per job before auto-skip. Prevents wasting 120s × N on consecutive timeouts.
Error Classification FIXED
3-tier error handling: transient (retry), permanent (skip), unknown (2 retries max).
